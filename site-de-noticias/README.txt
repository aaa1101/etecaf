Projeto desenvolvido em sala de aula para simular um site de notícias, onde as notícias devem possuir:
	- Título ( até 50 caracteres dependendo da fonte escolhida)
	- Linha fina (subtítulo com até 80 caracteres dependendo da fonte escolhida)
	- Uma foto
	- Nome do(s) autor(es)
	- Data e hora de publicação
	- Texto da notícia
	- Categoria

As notícias, quando exibidas na página do portal, devem somente exibir Título, Linha fina e Foto.
Quando acessada, deve ser exibida completa.

No caso do usuário clicar na categoria, a página deve somente exibir notícias da categoria
selecionada, voltando a exibir notícias mescladas ao voltar para a página inicial.
